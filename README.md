# Jigsaw Puzzle Game 🧩

A Windows Forms Jigsaw Puzzle Game built using 
C# and Object-Oriented Programming concepts.
Load any image and solve the puzzle!

---

## Features

- ✅ Load any image — JPG, PNG, BMP supported
- ✅ 3x3 puzzle grid — 9 tiles to arrange
- ✅ Smart shuffle — always solvable puzzle
- ✅ Move counter — tracks your moves
- ✅ Timer — tracks time taken to solve
- ✅ Original image shown on side for reference
- ✅ Win detection — congratulations message
- ✅ Resume button — pause and continue
- ✅ Clean OOP structure — 6 separate classes

---

## Project Structure

| File | Description |
|------|-------------|
| Tile.cs | Single puzzle piece |
| Board.cs | 3x3 grid logic |
| GameEngine.cs | Game state and timer |
| ImageSlicer.cs | Cuts image into 9 pieces |
| PuzzleForm.cs | Windows Forms UI |
| PuzzleForm.Designer.cs | UI components |

## How to Run

### Requirements:
- Windows OS
- Visual Studio 2022
- .NET Framework

### Steps:
1. Clone the repository:
git clone https://github.com/Mansoor-Mir1/JigsawPuzzleGame.git

2. Open Jigsaw.sln in Visual Studio 2022

3. Press F5 to run

---

## How to Play

1. Click **Load Image** to select any image
2. Click **Shuffle** to start the game
3. Click tiles to move them into empty space
4. Arrange all tiles to match original image
5. Win message shows your time and moves!
